# Media Folder — Drop Images Here

Drop your images into the right subfolder using the exact filenames below. The site will pick them up on next reload. **No upload needed — just save the file.**

## How it works

- The site looks for `.jpg`, `.png`, `.jpeg`, or `.webp` (in that order). Use whichever you have.
- If a matching file isn't found, the styled fallback art shows automatically.
- Recommended sizes: posters **400×600**, banners **1600×600**, characters **400×400** (square).
- Keep files under ~500KB each for fast loading.

## Folder structure

```
media/
├── posters/        ← homepage thumbnails (one per anime)
├── banners/        ← detail-page hero images (one per anime)
└── characters/
    └── {anime-id}/ ← portraits for that anime's characters
```

## Drag-and-drop in the browser also works

If you don't want to bother with files: just drag an image directly onto any poster, banner, or character avatar in the website. The image is saved in your browser (only visible to you on that device). Images dropped this way override the folder file. To revert, hover the image and click the **×** button in the top-right corner.
